Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag i (tag não semântica)

Nesta frase, temos um termo usando a tag em (tag semântica)

Texto marcado

Para marcar um trecho de um texto, se eu tiver vários marks, para mudar as cores de todos de uma vez eu vou até a área head e crio uma tag chamada style e daí dentro da style eu crio um seletor para todos os meus marks.

Texto grande e Texto pequeno

Para texto grande usamos a tag chamada big, porém essa tag já está obsoleta, para letras pequenas usamos a tag small, para deixar as letras pequenas.

Texto deletado

É o texto que eu risco porém ele está ali,para isso eu uso a tag del ,exemplo: Hoje é quarta-feira.

Texto inserido

Para o texto inserido usamos a tag ins, essa tag ela sublinha o texto, como se o texto que ali está estivesse preenchendo a página, exemplo: Hoje é quarta-feira.

Texto sobrescrito

Usamos a tag sup para texto sobrescrito,esse tipo de texto é para elevar uma letra ou número,como se eu estivesse fazendo uma conta,exemplo: 507+12,viu só, aparece um número elevado, que nesse caso é o 7.

Texto subscrito

Usamos a tag sub para texto subscrito,esse tipo de texto é para letra ou número, que eu quero menorzinho,exemplo: H2O, podemos ver que o 2 da fórmula da àgua ficou pequeno.
